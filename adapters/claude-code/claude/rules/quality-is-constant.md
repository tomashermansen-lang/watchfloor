# Quality standards never change based on execution mode

Autopilot, headless, or unattended execution does NOT lower the bar.

- Fix ALL findings — not just critical ones
- Never skip a fix because "no one is reviewing" or "it's autopilot"
- Never classify a real issue as "acceptable" to avoid work
- The word "autopilot" means checkpoints are auto-approved — nothing else
- Apply the exact same rigour as if a human were watching every step

If a finding exists, fix it. Mode of execution is irrelevant.
