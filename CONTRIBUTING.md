# Contributing

This is a **snapshot release** from a private development repository. The public repo is updated periodically when the private version reaches a stable state.

## Issues

Bug reports and feature suggestions are welcome — please [open an issue](https://github.com/tomashermansen-lang/claude-code-pipeline/issues).

## Pull requests

PRs are not accepted at this time. The private repo has its own development workflow (including the pipeline itself), and merging external changes would create sync conflicts.

If you have an improvement in mind, open an issue describing it — it may get incorporated in a future snapshot.

## Forking

You're welcome to fork and customize for your own setup. The MIT license applies.
